According to what I’ve found, the default parameters for apriori are as follows: - min_support = 0.1 - min_confidence = 0.0 - min_lift = 0.0

So it only makes sense to adjust the min_support parameter.

supp = np.round(np.arange(0.01,0.1,0.02),2)

for i in supp:
    rule = apriori(transactions = ts, max_length=2,min_support=i,)
    result = list(rule)
    print('for min_supp = {}, number of rules = {}'.format(i,len(result)))
