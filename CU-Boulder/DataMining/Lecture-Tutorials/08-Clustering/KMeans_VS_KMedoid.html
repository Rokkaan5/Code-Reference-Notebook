K-Means vs. K-Medoid

K-Means and K-Medoid

We learned both K-Means and K-Medoid as basic clustering methods. Let’s play with them and observe the differences.

Prepare the packages

  • K-Means is included in sklearn package, we can import it directly.
  • K-Medoid is included in sklearn extra package, we should install it first, then import it.
  • We will need numpy and pandas to create some dummy dataset
  • We will need seaborn for visualize the result.

import numpy as np
import pandas as pd
from sklearn.cluster import KMeans
from sklearn_extra.cluster import KMedoids
import matplotlib.pyplot as plt
import seaborn as sns

Round 1: Let’s create some simple dataset.

For visualization purpose, we create datapoints with 2 dimensions.

Create 3 clusters of data points.

Each cluster has 1000 data points, each data point has 2 dimensions. Data points has normal distribution with specified mean and std.

c1 = np.random.normal(5, 3, (100, 2))
c2 = np.random.normal(15, 5, (100, 2))
c3 = np.random.normal(-5, 2, (100, 2))
d = np.concatenate((c1, c2, c3), axis = 0)
d.shape
(300, 2)

Let’s visualize the dataset using Seaborn

sns.scatterplot(data = d, x = d[:,0], y = d[:,1])
<matplotlib.axes._subplots.AxesSubplot at 0x7f547096a2d0>

Let’s train a model using KMeans

kmeans = KMeans(n_clusters=3, random_state=0).fit(d)
sns.scatterplot(data = d, x = d[:,0], y = d[:,1],hue = kmeans.predict(d))
<matplotlib.axes._subplots.AxesSubplot at 0x7f546ef3fa10>

Lets train a model using KMedoid

kmedoids = KMedoids(n_clusters=3, random_state=0).fit(d)

Visualize the result

sns.scatterplot(data = d, x = d[:,0], y = d[:,1],hue = kmedoids.predict(d))
<matplotlib.axes._subplots.AxesSubplot at 0x7f54679ccc50>

Conclusion: KMean and KMedoid did equaly well for this dataset.

#Round 2: Let us add one outlier to the dataset

Create the same random clusters

c1 = np.random.normal(5, 3, (100, 2))
c2 = np.random.normal(15, 5, (100, 2))
c3 = np.random.normal(-5, 2, (100, 2))

Create an outlier.

The outlier is far away from all other data points

outlier = np.array([[100, 100]])
d = np.concatenate((c1, c2, c3, outlier), axis = 0)

Visualize the dataset

sns.scatterplot(data = d, x = d[:,0], y = d[:,1])
<matplotlib.axes._subplots.AxesSubplot at 0x7f54678f9a90>

Train a model using KMeans

kmeans = KMeans(n_clusters=3, random_state=0).fit(d)

Visualize the result

sns.scatterplot(data = d, x = d[:,0], y = d[:,1], hue = kmeans.predict(d))
<matplotlib.axes._subplots.AxesSubplot at 0x7f54678f6690>

Train a model using KMedoids

kmedoids = KMedoids(n_clusters=3, random_state=0).fit(d)

###visualize the result

sns.scatterplot(data = d, x = d[:,0], y = d[:,1],hue = kmedoids.predict(d))
<matplotlib.axes._subplots.AxesSubplot at 0x7f5467861390>

Conclusion: KMedoid did well even with an outlier. KMeans failed
