Data Visualization - Pandas

Data Visualization

It is hard to digest many data at the same time. Data visualizaiton is a way to make the process simple and straightforward.

For example, the sorting algorithms we have learned so far, are hard to beginners. However, a data visualizaiton can help the comparison much simpler: https://www.youtube.com/watch?v=kPRA0W1kECg

Because Data Visualization is so important and essential in today’s business, pandas as a data-driven package provides built-in plotting functions. We will learn some basic usage of them today.
